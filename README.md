DANILO 🎓
Digital Assistant Network for Interactive Learning Offline
Learning Offline): A Localized AI LMS for Disconnected Public Schools

🚀 Overview
Project DANILO addresses the digital divide by providing an offline-first educational platform. This repository contains the assests for project DANILO, featuring a futuristic easy access design, interactive lesson modules, and a gated progression system, As well as an offline
Integrated AI.

Key Features
Sequential Learning Path: Modules (Reading, Mathematics, Natural Sciences) are locked behind competency gates.

Integrated Assessments: Real-time quiz system with automated scoring and feedback using Offline AI.

Advanced UI/UX: Custom CSS implementation featuring:

- Dot-Grid Backgrounds and ambient glow effects.

- Inter Font Integration for high readability.

- Responsive Cards with hover states and status badges.

- Performance Analytics: A dedicated metrics dashboard to track curriculum completion and scores.
